# Conventions

## Naming
- Files: kebab-case (ProductDetail.tsx)
- Components: PascalCase
- Functions: camelCase

## Code Style
- TypeScript strict
- shadcn/ui components via Radix UI
- Tailwind for styling
- React hooks for state

## Patterns
- Services in `features/*/services/*.ts`
- Types in `features/*/types/*.ts`
- Contexts wrap providers in App.tsx
