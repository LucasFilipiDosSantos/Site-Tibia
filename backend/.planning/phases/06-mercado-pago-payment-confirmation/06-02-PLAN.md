<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| External Mercado Pago webhook -> API webhook endpoint | Untrusted internet payload can attempt privileged order/payment mutation. |
| API webhook ingress -> async processor queue | Validation/identity context crosses into deferred execution boundary. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-05 | S | Webhook sender authenticity | mitigate | Enforce canonical `x-signature` HMAC validation with app secret before enqueue/mutation (D-04/D-05). |
| T-06-06 | R | Duplicate/replayed notifications | mitigate | Persist dedupe key (`providerResourceId + action`) and short-circuit duplicates (D-06/D-07). |
| T-06-07 | T | Out-of-order event rollback | mitigate | Monotonic status guard: ignore/log regressions and block rollback writes (D-08). |
| T-06-08 | D | Webhook timeout causing retries/flood | mitigate | Fast ack pattern and defer heavy work to Hangfire worker path (D-13). |
| T-06-09 | R | Processing failure audit gaps | mitigate | Persist inbound log + processing outcome + dead-letter marker for terminal failures (D-14). |
</threat_model>
